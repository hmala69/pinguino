# Avance semana 2: Visión del proyecto Pingüino

La visión del proyecto Pingüino es desarrollar una placa de desarrollo basada en microcontroladores PIC, enfocada en facilitar el aprendizaje de electrónica, programación y sistemas embebidos.

Se busca que esta placa sea una herramienta accesible, práctica y fácil de utilizar para estudiantes, docentes y comunidades pequeñas que no siempre cuentan con recursos suficientes para adquirir placas comerciales o equipos más costosos.

La placa Pingüino no solo funcionará como una tarjeta de desarrollo para realizar prácticas con entradas, salidas, sensores, motores y comunicación, sino que también se plantea que pueda funcionar como un programador para microcontroladores PIC. Esto permitiría cargar programas directamente al PIC sin depender de herramientas externas más caras o difíciles de conseguir.

Con esta idea, el proyecto busca integrar en una sola placa una solución educativa completa: aprender, practicar, programar y probar circuitos con microcontroladores PIC de una manera sencilla y económica.

A futuro, se espera que el proyecto Pingüino pueda servir como una alternativa educativa frente a otras placas de desarrollo, promoviendo el aprendizaje práctico, el acceso abierto a la tecnología y el desarrollo de proyectos electrónicos en comunidades con recursos limitados.
