## 📌 Descripción del proyecto

**PINGÜINO UNO** es una placa de desarrollo diseñada con un microcontrolador **PIC**, pensada para facilitar el aprendizaje de electrónica, programación y sistemas embebidos.

El objetivo principal del proyecto es ofrecer una herramienta accesible, económica y fácil de usar para estudiantes, docentes y comunidades pequeñas que deseen aprender electrónica desde un nivel básico hasta aplicaciones más avanzadas.

---

## 🎯 Objetivo general

Desarrollar una placa de programación basada en un microcontrolador PIC que permita a los usuarios aprender, practicar y desarrollar proyectos electrónicos de forma sencilla, económica y accesible.

---

## 🚀 Características principales

- 🧠 Basada en microcontrolador PIC  
- 🔌 Fácil conexión para sensores y módulos externos  
- 💡 Ideal para prácticas escolares y proyectos educativos  
- 🛠️ Diseño enfocado en bajo costo y fácil fabricación  
- 📚 Pensada para comunidades con acceso limitado a tecnología  
- 🧪 Útil para aprender programación, electrónica digital y control básico  

---

## 🧩 Aplicaciones

Esta placa puede utilizarse para proyectos como:

- Encendido y control de LEDs  
- Lectura de botones y sensores  
- Control de motores  
- Comunicación con módulos externos  
- Prácticas de programación en microcontroladores  
- Proyectos escolares de electrónica  

---

## 🛠️ Tecnologías utilizadas

- Microcontrolador PIC  
- Diseño PCB  
- Electrónica digital  
- Programación embebida  
- GitHub para control de versiones
